ClipboardTyper Feature Overview

ClipboardTyper runs in your menu bar and types clipboard text into the active app after a global shortcut. Ideal for remote sessions where paste is blocked.

Report a Bug Send Feedback

Global Shortcut Trigger

Choose your own shortcut key and modifier combination from app settings.

Delay Before Typing

Set a countdown before typing starts so you can focus the target input field.

Delay Between Keys

Optional per-keystroke delay for remote apps that need slower synthetic input timing.

Feedback Built In

The app menu includes direct options for email feedback or opening a GitHub issue.

Permission Guidance

On first launch, the app prompts for typing permission and exposes quick accessibility settings access.

Remote Session Friendly

Uses virtual key events to improve behavior in environments like Citrix, RDP, and Parsec.
